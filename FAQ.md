# FAQ & Getting Help

## Frequently Asked Questions

### How do I start the stack locally?
See the [Developer Goodies](README.md#developer-goodies) section. Use `make up` or `docker compose up --build`.

### How do I add a new agent or operator?
See the [Operator README](apps/operator/README.md) and [Minimal Python Agent README](agents/python_agents/minimal_a2a_agent/README.md).

### Where are the API docs?
See [openapi.yaml](a2a_agent_umbrella/apps/a2a_agent_web/openapi.yaml) for the OpenAPI/Swagger schema, or [A2A Agent Web README](a2a_agent_umbrella/apps/a2a_agent_web/README.md).

### How do I run tests?
Use `make test`, `make test-elixir`, or `make test-python`. Manual commands are in the README.

### How do I get support or report a bug?
Open an [issue](https://github.com/<your-org>/hypergraph_agents_umbrella/issues) or join [GitHub Discussions](https://github.com/<your-org>/hypergraph_agents_umbrella/discussions).

### Where are the release notes and security policy?
See [CHANGELOG.md](CHANGELOG.md) and [SECURITY.md](SECURITY.md).
